# Homebrew Tap for Theda

Install the [Theda](https://github.com/oco-adam/theda) CLI:

```sh
brew install oco-adam/tap/theda
```

This installs:
- `theda` — the main CLI
- `create-theda-app` — the app scaffolder
- `theda-watch` — the file-watching dev loop

## Note

This formula currently builds from source, which requires OCaml tooling
to be installed during the build step. Pre-built bottles will be available
in a future release.
